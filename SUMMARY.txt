HOW IT WORKS:
------------
1. User selects a week (1-11)
2. Reads scenario-based question
3. Clicks an answer
4. IMMEDIATELY sees:
   - ✓ Green checkmark if correct
   - ✗ Red X if incorrect
   - Correct answer highlighted in green
   - Wrong answer highlighted in red
   - Detailed explanation
5. Moves to next question
6. Gets final score with performance breakdown
